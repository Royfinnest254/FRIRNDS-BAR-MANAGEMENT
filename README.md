# Friends Bar Management System

A comprehensive bar management system built with React, Vite, Cloudflare Workers, and Hono.

## Features

- **Inventory Management**: Track stock levels, pricing, and history.
- **Sales Tracking**: Record sales and monitor revenue.
- **Daily Stock**: Manage daily stock records and profit margins.
- **Reports**: Generate insights on sales and inventory.
- **Admin**: User management and system configurations.

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run local development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Deploy:**
   ```bash
   npm run deploy
   ```

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Cloudflare Workers, Hono, D1 Database
- **Auth**: Mocha Users Service
